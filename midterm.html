Q1. What does OOP stand for, and what are its four main principles?

OOP stands for Object-Oriented Programming. Its four pillars are: Encapsulation (bundling data and methods), Inheritance (child class inheriting from parent), Polymorphism (same method, different behaviors), and Abstraction (hiding complexity).

Q2. What is the difference between a class and an object?

A class is a blueprint or template. An object is an actual instance created from that blueprint. E.g., Car is a class; myCar is an object.

Activity 1

ATM Simulation Program

Built a Java ATM simulation that demonstrates encapsulation — user balance is stored as a private field and only accessed through methods like deposit(), withdraw(), and checkBalance(). The program uses conditional logic to validate transactions.

// ATM Simulation — Kezia Lorein Villegas public class ATM { private double balance; public ATM(double initialBalance) { this.balance = initialBalance; } public void deposit(double amount) { if (amount > 0) { balance += amount; System.out.println("Deposited: ₱" + amount); } } public void withdraw(double amount) { if (amount <= balance) { balance -= amount; System.out.println("Withdrawn: ₱" + amount); } else { System.out.println("Insufficient funds."); } } public void checkBalance() { System.out.println("Balance: ₱" + balance); } }
